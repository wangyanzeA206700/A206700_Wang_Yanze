Labs

Lab 1 - UI Design

Reflection: In Lab 1, I learned how to design a clean and intuitive user interface using Jetpack Compose. I focused on making the notes app easy to navigate for students, aligning with SDG 4 by reducing friction in study organization.

Lab 2 - Interaction

Reflection: Lab 2 taught me how to add interactive features like click listeners and text input validation. These features make My Notes more responsive and user-friendly, helping students quickly add and edit their study notes.

Lab 3 - Material Design

Reflection: In Lab 3, I applied Material Design principles to My Notes, improving the visual hierarchy and accessibility. This made the app easier to use for all students, supporting inclusive learning as outlined in SDG 4.

Lab 4 - Navigation

Reflection: Lab 4 focused on implementing navigation between screens in My Notes. I learned how to use Jetpack Navigation to create a seamless flow between the home screen, note list, and note editor, making study organization more efficient.

Lab 5 - Room Database

Reflection: In Lab 5, I integrated Room Database into My Notes to store notes locally. This ensures students' study materials are always accessible, even without internet, supporting continuous learning aligned with SDG 4.

Projects

Project 1 - My Notes

Problem Statement: Students often struggle to organize study materials across multiple platforms. My Notes provides a simple, offline-first solution to store and manage all notes in one place.

Lessons Learned: I learned how to combine UI design, interaction, and navigation into a complete app. I also improved my debugging skills and learned to prioritize features that directly benefit students.

Overall Reflection

From Lab 1 to Project 2, I have grown significantly as a mobile developer. I started with basic UI design and gradually built up to a full-featured notes app with local storage, navigation, and cloud integration. The biggest challenge was implementing Room Database and ensuring data persistence, but through trial and error, I learned to debug and optimize database operations. Throughout the course, I kept SDG 4 in mind, designing My Notes to be simple, accessible, and useful for students. This project has shown me how technology can support quality education by making study organization easier for everyone.
